# About the Project

**100 Pandas x Polars Puzzles** is a community-driven learning resource to help data practitioners transition from Pandas to Polars.

## The Goal
The primary goal is to make it easy for Pandas users to:
- Understand Polars syntax.
- Learn Polars concepts like expressions, contexts, and lazy execution.
- See direct side-by-side comparisons of common data manipulation tasks.

## The Format
Each puzzle:
1. Presents a small data problem.
2. Shows a solution in Pandas.
3. Shows an equivalent solution in Polars.
4. Explains the key idea behind the Polars approach.

*Note: The list of exercises is continually growing. Contributions, corrections, and improvements are always welcome!*
